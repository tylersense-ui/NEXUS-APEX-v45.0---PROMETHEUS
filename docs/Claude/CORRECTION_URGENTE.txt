PROBLÈME DÉTECTÉ:
  ❌ Erreur: this.ramMgr.getAvailableHosts is not a function
  
CAUSE:
  La méthode getAvailableHosts() n'existe pas dans RamManager.
  Le RamManager utilise getRamPools() qui retourne un objet
  avec une propriété "servers".

CORRECTION APPLIQUÉE (ligne 591-593):
  
  AVANT:
  const hostRAM = this.ramMgr.getAvailableHosts();
  
  APRÈS:
  const ramPools = this.ramMgr.getRamPools();
  const hostRAM = ramPools.servers || [];

COMPATIBILITÉ:
  ✅ getRamPools().servers retourne un tableau d'objets avec:
     - hostname
     - maxRam
     - usedRam
     - freeRam
     - utilization
  
  ✅ Le code utilise host.hostname et host.freeRam
  ✅ Totalement compatible avec l'API existante
