                      MES EXCUSES
═══════════════════════════════════════════════════════════════

Je suis désolé pour cette erreur de méthode. J'ai inventé une 
fonction qui n'existait pas au lieu de vérifier la documentation
du RamManager.

La bonne méthode est getRamPools() qui retourne un objet avec :
  - totalMax  : RAM totale du réseau
  - totalUsed : RAM utilisée
  - totalFree : RAM libre (c'est celle-ci qu'on veut)

Le patch corrigé ci-dessus devrait maintenant fonctionner 
parfaitement et saturer votre RAM à 95-100% comme vous le 
souhaitiez.
